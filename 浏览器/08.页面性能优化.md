# 页面性能优化

提高页面性能的方法：

- 资源压缩合并，减少 HTTP 请求，开启 Gzip 压缩

- 非核心代码异步加载

  - 异步加载方式
    - 动态脚本加载（用 Js 创建 script 标签）
    - defer：在 HTML 解析完之后才会执行，如果是多个，按照加载的顺序依次执行
    - async：在加载完成之后立即执行，如果是多个，执行顺序和加载顺序无关

- 利用浏览器缓存

  - 缓存分类

    - 强缓存
      - Expries
      - Cache-Control
    - 协商缓存
      - Last-Modified If-Modified-Since
      - Etag If-None-Match

  - 缓存原理

- 使用 CDN

- 预解析 DNS
